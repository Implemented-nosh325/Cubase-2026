1. Open the VST Organization menu.
2. Select the "Scan Plugins" button.

### Stem Export

Exporting stems requires precision. This tool automates the process of track separation and file naming.

1. Open your project in Cubase.
2. Open the Stem Export tab in Cubase-2026.
3. Select the folder where you want to save your stems.
4. Choose your audio format, such as WAV or FLAC.
5. Click "Start Export" to render all highlighted tracks as individual files at once.
